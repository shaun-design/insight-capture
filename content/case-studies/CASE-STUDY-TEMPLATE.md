# Case study content template

Copy this file, rename it (e.g. `my-study-draft.md`), fill every section, and paste the file or its contents back when you want it turned into code.

**Rules**

- Use **Title Case** for every `toc_label` (hero pills, right-hand rail, jump links).
- Keep `section_id` values exactly as shown (they power `#` links) unless you coordinate a code change.
- For bullets, use `- ` at the start of each line.
- Plain paragraphs: no special syntax needed; blank line between paragraphs.
